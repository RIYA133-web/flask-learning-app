# Flask Learning App

A full-stack CRUD web application built while learning Flask from scratch — 
covering routing, templates, forms, database operations, sessions, and 
Blueprint-based project structure.

## 🔗 Live Demo
[View Live App](https://flask-learning-app.onrender.com)

## 🛠️ Tech Stack
- **Backend:** Flask (Python)
- **Database:** SQLite + SQLAlchemy
- **Frontend:** Jinja2 Templates, Bootstrap
- **Deployment:** Render

## ✨ Features
- User form with CRUD operations (Create, Read, Update, Delete)
- Session-based flash messages for user feedback
- Blueprint-based modular project structure
- Deployed end-to-end with Gunicorn

## 📁 Project Structure

flask-learning-app/
├── app.py
├── models.py
├── routes/
│   └── users.py
├── templates/
├── requirements.txt
└── Procfile


## 🚀 Getting Started
```bash
git clone https://github.com/RIYA133-web/flask-learning-app.git
cd flask-learning-app
pip install -r requirements.txt
python app.py
```
